This website is hosted on an old but fast  Pentium 4 server that I brought back to life with Ubuntu. 
On this server I've installed Apache, ssh, and mod_security to prevent attacks to the server.
I've used a free domain name service with CloudFlare as DNS provider to speed up my website and prevent attacks. I've also set up a few subdomains just in case (smiley): If you need one to host your website, just send me a message (smiley)(smiley)(smiley). Plus, I've used a php script and CloudFlare's geolocation option to automatically redirect the user to the Italian, Russian or English version of the website.
